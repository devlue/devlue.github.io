HTML / Markdown 자주쓰는 특수문자 표기법입니다.

**사용하실 때 문자 끝에 ;(세미콜론) 표기하는거 잊지마세요.**

| 표현문자 | 문자표현 |
| :---: | :---: |
| " | &quot |
| & | &amp |
| < | &lt | 
| > | &gt | 
| Non-breaking space (띄어쓰기) | &nbsp |  
| ¡ | &excl |
| ￠ (센트) | &cent | 
| ￡ (파운드) | &pound |
| ￥ (엔) | &yen |
| ⓒ (저작권) | &copy |
| ± | &plusmn |
| ² | &sup2 |
| ³ | &sup3 |
| ° | &deg |
| ½ | &frac12 |
| ¼ | &frac14 |
| ¾ | &frac34 |
